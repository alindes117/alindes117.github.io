Explore-It! Creative Brief

  • Title: Flying a Plane
  • Project Summary: This project will demonstrate and educate the user on how to fly a plane. It will cover the controls of a plane, such as rolling, pitches, yawing, and reading the radar display.
  • Audience Profile: Kids between the ages of 6-12 years old will be using this website.
  • Perception and Tone: Kids should learn the basics of flight control and understand how to fly a plane through direct interactions.
  • Communication Strategy: In order to create an engaging experience, I'll create an interactive site that works on all screen sizes. Users will be able to interactive through click and touch inputs.
  • Target Message: Planes can be intimidating but taking the time to understand each part will hopefully develop an interest in the user.
